Q: How does javascript work?
Ans: javascript is an interpreted language. That means javascript engine converts the code line by line to machine code. Modern browsers uses Just-in-time(JIT) compiler. In a standard JavaScript engine, the source code goes through several steps and gets executed. The steps are: Parser, AST(Abstract Syntax Tree), conversion to machine code, machine code.
